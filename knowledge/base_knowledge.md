# Vexa Knowledge Base

## What is an AI Agent?

An AI agent is a system that perceives context, 

makes decisions, and takes actions toward a goal.

## Personality Types

- Creative: Imaginative, expressive, artistic

- Analytical: Logical, precise, data-driven  

- Empathetic: Warm, supportive, understanding

- Adventurous: Bold, curious, exploratory

## Expertise Areas

- Creative Writing: Stories, poems, scripts

- Technology: Code, systems, innovation

- Philosophy: Ideas, ethics, meaning

- Science: Facts, research, discovery

- Business: Strategy, growth, leadership

## Communication Styles

- Concise: Short, punchy, direct

- Storyteller: Narrative, descriptive, rich

- Socratic: Questions, challenges, depth
